# Advancing satellite solar-induced chlorophyll fluorescence data assimilation in land surface models: An assessment over boreal evergreen needleleaf forests

## Overview
This repository contains the Python notebooks used to produce the figures in the manuscript:
**Advancing satellite solar-induced chlorophyll fluorescence data assimilation in land surface models: An assessment over boreal evergreen needleleaf forests**
L. Leverne, C. Abadie, C. Bacour, V. Tartaglione, Y. Zhang, T. A. Black, V. Bastrikov, N. Raoult, P. Peylin, A. Krieger-Liszkay, and F. Maignan
in revision for *Journal of Advances in Modeling Earth Systems*

## Requirements
- Python 3.9

## Data
The csv files are provided. The netcdf files are available upon request to the authors.
